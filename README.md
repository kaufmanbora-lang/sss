# Заметка → экран блокировки

Интерактивное полноэкранное PWA на основе двух исходных фотографий. Заголовок вводится в реальное поле, галочка открывает фотографический экран блокировки, а введённый текст появляется над нижними кнопками.

## Запуск

```powershell
pnpm install
pnpm dev
```

Для локальной проверки production-сборки:

```powershell
pnpm build
pnpm preview
```

На iPhone откройте опубликованный HTTPS-адрес в Safari → «Поделиться» → «На экран “Домой”». Manifest и метатеги настроены для полноэкранного запуска без браузерной панели.

На телефоне поле открывает настоящую системную клавиатуру устройства. Точная фотографическая панель Notes автоматически размещается непосредственно над клавиатурой по данным `visualViewport`. На компьютере можно вводить текст физической клавиатурой. Escape или двойное нажатие на экран блокировки возвращает к заметке.

## Публикация на GitHub Pages

1. Создайте пустой репозиторий на GitHub.
2. Загрузите в корень репозитория всё содержимое этого проекта, включая скрытую папку `.github`.
3. Убедитесь, что основная ветка называется `main`.
4. В `Settings → Pages → Build and deployment → Source` выберите `GitHub Actions`.
5. Откройте вкладку `Actions`: workflow `Deploy to GitHub Pages` автоматически соберёт и опубликует сайт.

При следующих изменениях новый деплой будет запускаться после каждого push в `main`. Путь GitHub Pages вида `/имя-репозитория/` поддерживается благодаря относительным URL сборки.

## Публикация на Render

В корне уже находится `render.yaml` для бесплатного Static Site.

1. Сначала загрузите проект в GitHub.
2. Откройте `https://dashboard.render.com/blueprint/new`.
3. Подключите созданный репозиторий и примените найденный Blueprint.
4. Render выполнит `pnpm build` и опубликует папку `dist`.

База данных, секреты и переменные окружения этому сайту не нужны. Render автоматически обновляет сайт после push в ветку `main`.

После публикации сайт находится на серверах Render/GitHub и работает независимо от вашего компьютера. Для установки PWA используйте именно выданный HTTPS-адрес.
